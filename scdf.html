Spring Cloud Data Flow

Build dynamic and real time data flow pipelines with micro-services in the cloud that integrates disparate systems. Turn your spring integration components into spring cloud stream and spring batch into spring cloud task micro-services.

Overview

We have been implementing spring integration based data flow pipelines since many years. Also we have been writing spring batch based applications to implement batch processing applications. But then these were all monoliths that did scale but not very efficiently.

With Spring Cloud Data Flow, you can now deploy each spring integration application as a micro-service along with its IntegrationFlow. Also, each spring batch component is a micro-service. Moreover, you can build real-time data pipelines by selecting source, processor & sink components on the fly.

You may deploy your applications on top of K8s, Apache Mesos, Yarn or Cloud Foundry.
